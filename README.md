**Live demo**
[zdenekjon.cz](https://the-el0.github.io/zdenekjon)


--



## Get started

### Download from GitHub
```
$ git clone https://github.com/The-El0/zdenekjon.git
$ cd git/zdenekjon
$ npm install
```


### Build SASS (run first time only)
```
$ npm run build
```


### Start local server && live reload when HTML and SASS changes
```
$ npm run start
```
